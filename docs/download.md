# YaS☀️lR Downloads

**Please make sure to download the firmware matching your board.**

Firmware files are named as follow:

- `YaSolR-<VERSION>-<MODEL>-<BOARD>.bin`: for the firmware used to update
- `YaSolR-<VERSION>-<MODEL>-<BOARD>.factory.bin`: for the firmware used to flash for the first time or to do a factory reset

Where:

- `VERSION`: YaS☀️lR version, or `main` for the latest development build
- `MODEL`: `oss`, `pro` or `trial`
- `BOARD`: the board name, from the list of [Compatible ESP32 boards](/build#compatible-esp32-boards)

## Open-Source version

- [latest](https://github.com/mathieucarbou/YaSolR-OSS/tree/main/latest) (latest development build can be unstable)

_Firmware and source code for the Open-Source version are available directly in the GitHub project at [https://github.com/mathieucarbou/YaSolR-OSS/releases](https://github.com/mathieucarbou/YaSolR-OSS/releases)._

## Pro version

- [latest](https://github.com/mathieucarbou/YaSolR-Pro/tree/main/latest) (latest development build can be unstable)

_Firmware for the Pro version are available at [https://github.com/mathieucarbou/YaSolR-Pro](https://github.com/mathieucarbou/YaSolR-Pro)._
_This location is only available to Pro users and you must be logged into your GitHub account to access it._
